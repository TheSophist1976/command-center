## REMOVED Requirements

### Requirement: CreateNote NLP action
**Reason**: The NLP module is removed entirely. NLP-driven note creation is no longer supported.
**Migration**: Use the `a` key in Notes view to create notes manually.

### Requirement: EditNote NLP action
**Reason**: The NLP module is removed entirely.
**Migration**: Use the note editor directly.

### Requirement: NLP system prompt includes note instructions
**Reason**: The NLP module and system prompt are deleted.
**Migration**: No replacement.

### Requirement: NLP note context
**Reason**: The NLP module is deleted entirely.
**Migration**: No replacement.
